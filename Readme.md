# Manga 
## A microservice for latest manga from various sources.

[![Heroku App Status](https://heroku-shields.herokuapp.com/manganode)](https://manganode.herokuapp.com)

## Services working :

```
- manga/0/hot-updates
- manga/0/latest-updates
- manga/0/get-all
- manga/0/search/?keyWord=<query>
```
